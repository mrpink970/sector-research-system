📝 Manual Edit Instructions

To OPEN a trade:
1. Fill out form above → Click "Open Trade"
2. CSV line copied to clipboard
3. Go to GitHub → Open data/ttp/trades.csv
4. Paste the new line at the bottom
5. Commit changes

To CLOSE a trade:
1. Click "Close Trade" → Enter exit price
2. CSV line copied to clipboard
3. Go to GitHub → Find the open trade line in trades.csv
4. Replace it with the completed line from clipboard
5. Commit changes

CSV Format:
ticker,entry_date,entry_price,shares,stop_price,target_price,status,exit_date,exit_price,profit
